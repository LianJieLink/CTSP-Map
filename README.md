# CTSP Map Generator

這是一個基於 Python 與 Folium 的地圖生成腳本，主要用於視覺化科學園區 (包含中科、竹科、南科等) 周遭的重要設施與環境監測站。

## 功能

執行 `CTSP-map.py` 後，將會產出一份互動式 HTML 地圖檔 (`index.html`)，其中包含以下圖層與標示：

1. **底圖**：
   - 預設提供 OpenStreetMap 街道圖與 Google 衛星影像，可自由切換。
2. **廠區範圍 (紅色多邊形)**：
   - 台積電 15A 廠、15B 廠、AP5 廠、AP3 廠、AP6 廠、F22 廠
   - 台中零廢製造中心
3. **測站位置**：
   - **環境部測站 (藍色星號)**：包含沙鹿、西屯、忠明、楠梓、左營、仁武、龍潭、新竹、頭份、朴子、嘉義、新港、善化、安南等站。
   - **特殊性工業區測站 (綠色三角形)**：涵蓋中科 (陽明國小、國安國小、都會公園、實驗中學)、竹科 (龍潭、德龍、聖德、新安、靜心、力行、興業、南東、南西、南南、南北)、南科 (公13、公19、公29、實中)，以及宏毅社區(空品車)與六輕東石等測站。
   - **氣象署測站 (橘色雲朵)**：包含大雅(自動)、高雄(人工)、龍潭(自動)、新竹市東區(自動)、寶山(自動)、竹東(自動)、頭份(自動)、龍鳳(自動)、六腳(自動)、蔦松(自動)、善化(自動)、安定(自動)。
4. **互動元件**：
   - 圖層控制開關 (Layer Control)
   - 全螢幕顯示 (Fullscreen)
   - 游標位置經緯度顯示 (Mouse Position)
   - **距離與面積量測工具 (Measure Control)**

## 依賴套件

請確保你的環境已安裝 `folium`：

```bash
pip install folium
```

## 執行方式

在終端機中執行指令：

```bash
python CTSP-map.py
```

執行完畢後，會在同個目錄下生成 `index.html`。你可以直接透過網頁瀏覽器開啟此檔案以瀏覽互動式地圖。
